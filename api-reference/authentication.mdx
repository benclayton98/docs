# Authentication

Obtain a Bearer token by calling `POST /auth/token` with **HTTP Basic Auth** (`client_id:client_secret`).  
Tokens expire every 24 hours by default. You may also enable a **replicable per-day** token mode server-side.

## Get a token

### cURL
```bash
curl -X POST https://api.housr.com/auth/token \
  -u "your_client_id:your_client_secret" \
  -H "Content-Type: application/json" \
  -d '{"daily": false}'
